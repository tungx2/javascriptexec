자바스크립트에서 문서 출력하는 방법

자바스크립트에서 다양한 방식으로 데이터 출력하기

  1. document객체에 write메서드를 사용 - document.write('출력할 내용');
  2. window객체를 활용하여 메세지 띄우기 - window.alert('메세지 내용');
  3. 브라우저 개발자모드의 console창에서 출력하기 - console.log(출력할 내용);
  4. document객체의 innerHTML메서드로 내용 출력하기 - document.getElement('id명').innerHTML="출력할 내용";
