javascript_객체(window, 브라우저)

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

Window 객체

window객체는 브라우저 객체 중 최상위 객체로서 현재 실행되고 있는 브라우저 창을 나타낸다.

window객체의 메소드 종류
alert() - 윈도우 창에서 경고창(메세지)를 띄울 때 사용
prompt() - 사용자가 데이터를 입력할 수 있는 명령프롬프트창.
confirm() - 질문내용에 대한 확인, 취소를 하고자 할 때 사용.(true,false값)
setInterval() - 일정시간동안 작성한 코드나 함수를 실행하고자 할 때 사용
clearInterval - 일정시간동안 설정한 함수나 코드에 적용된 시간을 제거할 때 사용
open() - url주소를 삽입하여 현재 페이지가 아닌 새로운 창에서 페이지가 나오도록 함.
close() - 윈도우창을 닫을 때 사용
