jquery_slider(swiper)스와이퍼 슬라이드

http://swiperjs.com 스와이퍼 슬라이드 플러그인 - 모바일 터치 통작에 최적화 되어있으며 여러가지 형태의 슬라이드를 제작할 수 있다

작성방법
1. css폴더 안에 swiper.css 문서를 저장하고 html문서에 link삽입한다.
2. script 폴더안에 swiper.js 문서를 저장하고 html 문서에 script를 삽입한다.
3. 슬라이드를 삽입하기 위한 영역에 div class="swiper-container" 태그를 영역으로 만든다.
4. swiper-container이름 옆에 클래스명 하나 더 추가해서 만든다. 슬라이드가 여러개일 경우 겹치지 않게 하기 위함.
5. container 안에 .swiper-wrapper영역을 만든다.
6. .swiper-wrapper 영역 안에 들어갈 이미지 영역을 .swiper-slide로 만든다.
7. 좌,우 방향버튼을 삽입시 div class="siwper-button-next", div class="swiper-button-prev "
8. 컨트롤 버튼 삽입시 div class="swiper-pagination"을 마지막 /div 위에 삽입한다.
9. body아래 마지막 줄에 script소스 코드를 넣어서 슬라이드가 움직이게 만든다.
