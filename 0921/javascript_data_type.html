javascript_data_type(데이터 유형)

자바스크립트에서 사용하는 데이터 유형에 대해 알아보기

변수 안에 들어간 데이터의 유형을 알아보기 위해서는 typeof 연산자를 활용하면 된다.

  • 문자데이터형 (string)
  • 숫자데이터형 (number)
  • 부울형 (boolean) - true(확인),false(취소)
  • 찾을 수 없는 데이터형 (undefinded)
  • 값이 없는 데이터 (null)
  • 배열형 데이터 (array)
  • 함수형데이터 (function)
  • 객체 (object) - document, window, alert, confirm
